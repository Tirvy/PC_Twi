Директивы:
	newevent (id, list) - создает ивент с именем id в спписке ивентов list.
	return (retstatus) - завершиет описание ивента с возвращаемым знаением. (-1) - БАГА!; (0) - оставить; (1) - удалить.
	int - создать переменную. Потом обращаться к этой переменной, ставя перед названием @. По умолчанию - 0.
	foreach - для каждого
		возможные параметры: $Planet, $Player, $System, $Army, $Planet
		тело foreach, до endloop, выполняется для каждого $
	if () - если
		если значение в скобках >0, то входит в тело.
	else - выполняет, если предыдущий if был <=0.
	endif - заканчивает тело if (самого верхнего)
Переменные:
	$ActivePlayer - игрок, на чьем ходу/для которого выполняется ивент. Также все следующее применимо и для $Player и $SelectedPlayer
		.HS - домашняя система игрока
		.SAC - strategy allocation
		.FSC
		.CPC
		.Initiative - инициатива игрока
		.Stats
	$Planet, $SelectedPlanet
		.Owner - $Player, который контролирует
		.Amry - армия на этой планете
		.Tech - цвет техноогии планеты (1красный, 2синий, 3зеленый и 4желтый)
		.Res - ресурсоемкость планеты
		.Inf - Влияние планеты
		.EXH - 1, если exhausted, иначе - 0.
		.System
	$System, $SelectedSystem
		.id - имя системы
		.Owner - Контролирующий игрок
		.Fleet - флот в системе
		.WormHoles
Ситуационные переменные:
	Fight: $CurSystem, $CurPlanet, $CurAttacker, $CurDefender
Функции:
	Distance - расстояние между системами.
	Spawn (Type,N)<...> [owner,system]- тип/кол-во юнитов, хозяин, система
	Select - Planet, System, Player. Устанавливает $Selected*.
Константы:
	Mecatol - планета мекатол рекс
Операции:
	+, =, -, *, /, %; ++, --; <, >, ==, >=, <=, !=;
